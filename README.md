# Time Series Modelling using Hidden Markov Models (HMM)
This project applies Hidden Markov Models (HMM) to identify latent regimes and structural changes in time series data.

## Key features
- Seasonality adjustment (STL decomposition)
- Gaussian HMM estimation (state-dependent parameters)
- Model selection (AIC, BIC, log-likelihood)
- Regime detection and interpretation

## Results
The model identifies latent regimes but shows limitations in capturing smooth long-term trends.

## Tools
R (depmixS4, tidyverse, ggplot2)

## Objective
Explore regime-switching models and their application to non-stationary time series.
